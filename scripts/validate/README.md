# Scripts (Operator Tooling)

This directory contains **operator-only helper scripts** used for:
- Local testing
- Sandbox resets
- Installation verification
- Phase certification and validation

⚠️ These scripts are **not executed by the Task Assistant GitHub App**  
⚠️ These scripts are **not part of the Marketplace offering**

Marketplace installations interact with Task Assistant **only** via GitHub App workflows and configured automation.
